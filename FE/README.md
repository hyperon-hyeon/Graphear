# Graphear FE 계획표
시각장애 학생의 학습 보조를 위한 서비스「그래피어(Graphear)」 프론트엔드 개발 계획표

## 주차별 계획

| 주차 | 개발 내용 | 세부내용|
|---|---|---|
| 1주차 | 메인 홈 페이지 구현 | 로고 페이지, 메인 선택 페이지|
| 2주차 | 선택 페이지 / PDF 파일 변환 페이지 구현 | 학교, 학년, 과목, 단원 선택 페이지|
| 3주차 | 문제 재생 페이지 구현 | 문제 오디오 재생 페이지 구현 |
| 4주차 | 최종 API 연결 | 페이지 세부 점검 및 API 연결|
